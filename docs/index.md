# ToonPrompt

ToonPrompt is a CLI and Python SDK that compresses structured prompt payloads into a Toon-style representation before forwarding requests to coding assistants.

## Highlights

- Structured-only compression with fail-open behavior
- Tool wrappers for Codex, Claude, Cursor, and Gemini
- Optional audit logs and local metrics
- Extensible compressor plugin interface
- Python SDK for direct application integration
