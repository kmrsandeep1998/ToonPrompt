# Quick Start

## Install

```bash
pip install toonprompt
```

or with `pipx`:

```bash
pipx install toonprompt
```

## Initialize config

```bash
toon config init
```

## Inspect a prompt

```bash
toon inspect --prompt-file prompt.txt --preview --explain
```

## Dry run a wrapped tool

```bash
toon codex --prompt-file prompt.txt --dry-run
```
