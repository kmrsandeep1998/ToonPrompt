# CLI Reference

Top-level:

- `toon inspect`
- `toon codex|claude|cursor|gemini`
- `toon metrics`
- `toon audit`
- `toon check`
- `toon doctor`
- `toon config init`
- `toon version`

Useful flags:

- `--profile <name>`
- `--verbose` / `--quiet`
- `--dry-run`
- `--preview`
- `--explain`
- `--diff` (inspect)
- `--segment N` (inspect)
- `--format text|json|markdown` (inspect)
