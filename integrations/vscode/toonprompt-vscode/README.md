# ToonPrompt VS Code Extension (Starter)

This is a starter scaffold for the separate `toonprompt-vscode` repository.

## Planned features

- Command: `ToonPrompt: Optimize Selection`
- Status bar token estimate for active file
- Settings for ToonPrompt binary path and profile

## Local development

```bash
cd integrations/vscode/toonprompt-vscode
npm install
npm run compile
```

Open this folder in VS Code and press `F5` to launch an Extension Development Host.
